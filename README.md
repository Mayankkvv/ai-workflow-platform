# AI Workflow Automation Platform

A full-stack platform for visually building and running automated workflows —
similar in concept to tools like Zapier or n8n — powered by drag-and-drop
nodes and the Groq AI API.

> 🚧 This project is under active development. Progress is being built
> incrementally, step by step, with a focus on production-grade architecture.

## Tech Stack

**Frontend:** React, Vite, Tailwind CSS, React Flow, Zustand
**Backend:** Node.js, Express, MongoDB, Redis, BullMQ, Socket.IO
**AI:** Groq API

## Status

Currently in early development — project scaffolding and environment setup.
